Gladia's API can generate translations of your transcriptions into one or more target languages. You can translate your transcription into multiple languages in a single API call.
 
The list of the languages covered by the translation feature is slightly different than the supported languages for transcription. You can see the supported language [here](https://docs.gladia.io/api-reference/v2/pre-recorded/init#body-translation-config-target-languages).

If [subtitles](/chapters/pre-recorded-stt/captions) and/or [sentences](/chapters/settings/formatting#sentences) are enabled, the translations will also include translated results for these features.


## Usage

The translation is configured in the `translation` and `translation_config` keys in your transcription request. API reference is available [here](https://docs.gladia.io/api-reference/v2/pre-recorded/init#body-translation).

```json
{
  "audio_url": "<your audio url>"
  "translation": true,
  "translation_config": {
    "target_languages": ["en"],
    "model": "base"
  },
}
```

The `translation_config` object supports the following options:

| **Option**                | **Description**                                                                                                                                  |
|---------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| target_languages          | Array of language(s) to get the translation                                                                                                      |
| match_original_utterances | Align translated utterances with the original ones                                                                                               |
| model                     | Translation model to use <br/>  - `base` : Fast, cover most use cases <br/> - `enhanced` : Slower, but higher quality and with context awareness |


## Results

Your transcription result will have a `translation` key containing your translation(s) in the `results` object.

```json
{
  "transcription":{...},
  "translation": {
    success: true,
    is_empty: false,
    results: [
      {
        words: [
          {
            word: "Diviser",
            start: 0.20043,
            end: 0.7008000000000001,
            confidence: 1
          },
          {
            word: "l'infini",
            start: 0.9009500000000001,
            end: 1.5614400000000002,
            confidence: 1
          },
          ...
        ],
        languages: [Array],
        full_transcript: "Diviser l'infini dans un temps où moins est plus...",
        utterances: [Array], // Also translated
        error: null
      },
      {
        words: [
          {
            word: "Dividir",
            start: 0.20043,
            end: 0.7008000000000001,
            confidence: 1
          },
          {
            word: "la infinidad",
            start: 0.9009500000000001,
            end: 1.5614400000000002,
            confidence: 1
          },
          ...
        ],
        languages: [Array],
        full_transcript: "Dividir la infinidad en un tiempo en que menos es más, donde demasiado nunca es suficiente...",
        utterances: [Array], // Also translated
        error: null
      }
    ],
    exec_time: 0.6475496292114258,
    error: null
  }
}
```
