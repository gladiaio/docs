## Example configuration
```json request data
{
  "audio_url": "YOUR_AUDIO_URL",
  "custom_vocabulary": true,
  "custom_vocabulary_config": {
    "vocabulary": [
      "Westeros",
      {"value": "Stark"},
      {
        "value": "Night's Watch",
        "pronunciations": ["Nightz Vatch"],
        "intensity": 0.4,
        "language": "de"
      }
    ],
    "default_intensity": 0.6
  }
}
```
