<Note>
  If you don't know which language you have please use the [Automatic Single Language Detection configuration](/chapters/pre-recorded-stt/features#automatic-language-detection).
</Note>

In our API configuration and results, language codes follow ISO 639-1 (two-letter) where available. When no 639-1 code exists, we use the corresponding ISO 639-3 three-letter code (e.g., `yue` for Cantonese, `haw` for Hawaiian).
