<Note>
  Not sure which language is spoken ? See [Automatic language
  detection](/chapters/language/language-detection). For conversations mixing
  languages, see [Code switching](/chapters/language/code-switching).
</Note>

In our APIs and results, languages use 2-letter ISO 639-1 codes. When no 639-1 code exists, we use the corresponding ISO 639-3 three-letter code (e.g., `yue` for Cantonese, `haw` for Hawaiian).

<Tip>
  Use the 2-letter code in <code>language_config.languages</code> (e.g.{" "}
  <code>["en", "fr"]</code>).
</Tip>
