- with cloned speaker voice
only one channel supported
{
    cloned_speaker_voice : {
        data: base64 audio encoded data,
        sample_rate: sample rate of the encoded audio
    }
}

start sending text and language data

{
    text:'i love you',
    language:'en'
}

supported languages are
    EN = "en"
    ES = "es"
    FR = "fr"
    DE = "de"
